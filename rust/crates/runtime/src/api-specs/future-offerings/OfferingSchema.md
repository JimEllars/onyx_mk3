# Offering Schema

## Ebooks & Digital Courses

When ingesting "Transactional Feedback" and "Content Quality" for non-software products like ebooks and digital courses:

- Analyze feedback to determine if the course structure needs adjusting.
- If negative feedback is >10%, consider an autonomous marketing pause or trigger an EscalateToAdmin.
- Content Quality must be scored out of 5 based on customer sentiment analysis.
- Prepare recommendations for product updates based on this analysis.
